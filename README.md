# Open Research Audit (ORA)\n\nAn open-source multi-agent research tool that audits its own findings. Built with LangChain and LangGraph.\n\n**ORA**: Open Research Audit. Submit a query with an intensity level, get back calibrated, source-traced research with a built-in adversarial reviewer.\n\n## Status\n\nPre-alpha. Spec and design in progress.\n\n## Key Differentiators\n\n- **Adversarial reviewer** -- separate agent with fresh context attacks findings, verifies URLs, searches for contradicting evidence\n- **Calibrated confidence** -- every claim has an explicit confidence level (High/Moderate/Low/Unknown)\n- **Source traceability** -- claims traced back to root sources with corroboration tracking\n- **Intensity levels** -- configurable depth from quick check to exhaustive research\n- **Benchmark-validated** -- scored against Deep Research Bench
